How It Works

ExoHabitAI analyzes key parameters to predict habitability using a machine learning model with 94% accuracy. The system evaluates factors such as planetary mass, orbital distance, stellar temperature, and atmospheric potential.

The model considers multiple parameters including:

  • Planetary mass and composition
  • Orbital characteristics and distance from host star
  • Stellar temperature and radiation levels
  • Atmospheric retention capability
  • Potential for liquid water existence
